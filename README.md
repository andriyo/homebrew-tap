# ShadowDroid Homebrew Tap

Install ShadowDroid on macOS or Linux:

```sh
brew install andriyo/tap/shadowdroid
```

ShadowDroid requires Android Platform Tools (`adb`) on PATH:

```sh
brew install --cask android-platform-tools
```

Upgrade later:

```sh
brew upgrade shadowdroid
```

ShadowDroid releases are published from:

https://github.com/andriyo/ShadowDroid/releases
